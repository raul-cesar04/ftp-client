# ftp-client
FTP Client built in java using Apache Commons Net.

-[x] Arquivos

  -[x] Upload de arquivos para o Servidor.
  -[x] Remover Arquivos. 
  -[x] Renomear arquivos.
  -[x] Listar arquivos. 
-[x] Diretórios
  -[x] Criar diretórios.
  -[x] Remover Diretórios.
  -[x] Renomear Diretórios.
  -[x] Listar Diretórios.
-[ ] Miscelânea
